# 底盘控制项目 — 知识点与高阶思想总结

## 项目概述

基于 TI MSPM0 系列 MCU（MSPM0G3507，ARM Cortex-M0+）的 FreeRTOS 嵌入式项目，实现了双轮差速小车的**灰度传感器循迹**功能。

---

## 1. 双闭环级联 PID 控制架构

```
┌──────────────────────────────────────┐
│  外环：循迹控制 (50Hz)               │
│  输入：灰度偏置 → lineFollowPID(P)    │
│  输出：差速 RPM 偏移量                │
└──────────────┬───────────────────────┘
               ▼ 目标速度 + 低通滤波
┌──────────────────────────────────────┐
│  内环：速度控制 (100Hz)              │
│  输入：编码器实测RPM → MotorPID(PI)  │
│  输出：PWM 占空比 + 低通滤波         │
└──────────────────────────────────────┘
```

**核心思想**：外环只负责"方向决策"（纯 P，Kp=1），内环负责"精确执行"（PI，Kp=0.01, Ki=1e-3）。利用**频域分离**——外环低频修正方向，内环高频跟踪速度，避免两个 PID 互相打架。

**关键代码**：
- 外环 PID：[main.cpp#L233-L238](Sources/main.cpp#L233-L238)
- 内环 PID：[main.cpp#L82-L104](Sources/main.cpp#L82-L104)
- 速度更新：[Motor.hpp#L32-L41](Libraries/Motor/Motor.hpp#L32-L41)

## 2. 多层低通滤波链

项目中使用了**三级串联滤波**来消除抖动，这是稳定性好的根本原因：

| 层级 | 位置 | 算法 | 作用 |
|------|------|------|------|
| 空间滤波 | [main.cpp#L49-L71](Sources/main.cpp#L49-L71) | 7 传感器加权平均 | 消除单个传感器噪声 |
| 目标滤波 | [Motor.hpp#L24-L28](Libraries/Motor/Motor.hpp#L24-L28) | 一阶 IIR (α=0.6) | 速度指令不突变 |
| 输出滤波 | [Motor.hpp#L36-L38](Libraries/Motor/Motor.hpp#L36-L38) | 一阶 IIR (α=0.8) | PWM 变化平滑 |

```cpp
// 一阶惯性滤波通用公式：y[n] = α·x[n] + (1-α)·y[n-1]
expectedRPM = rpm * 0.6f + expectedRPM * 0.4f;   // α=0.6
feedback   = pid.calc(...)*0.8f + prevFeedback*0.2f; // α=0.8
```

## 3. 混合式控制策略：PID 闭环 + 开环状态机

**直线/缓弯用 PID 闭环，直角弯用开环预设序列**——这是嵌入式竞赛/小车项目中的经典策略：

- 闭环处理**可预测**的连续偏差 → 省参数、稳
- 开环处理**突变**的离散事件 → 不需要调参、确定性强

触发直角弯的条件检测：[main.cpp#L260](Sources/main.cpp#L260)：
```cpp
// 最左侧两个传感器同时黑 + 最右侧白
(graySensor_GPIOValues & 0b01) && (graySensor_GPIOValues & 0b10) && (~graySensor_GPIOValues & 0b1000000)
```

四段式转弯实际上是一个手工调试的**梯形速度剖面**：入弯加速 → 急转 → 收弯减速 → 直行回线。

## 4. 差分驱动运动学

两轮差速底盘的核心公式：[main.cpp#L137-L141](Sources/main.cpp#L137-L141)：

```cpp
motor1.setExpectedSpeed( commonSpeed + diffSpeed/2 );  // 左轮
motor2.setExpectedSpeed(-commonSpeed + diffSpeed/2 );  // 右轮（反向安装）
```

- `diffSpeed > 0` → 左轮快、右轮慢 → 右转
- `diffSpeed < 0` → 左轮慢、右轮快 → 左转
- `diffSpeed = 0` → 两轮等速反向 → 直行

## 5. 有限状态机 (FSM)

[main.cpp#L16-L23](Sources/main.cpp#L16-L23) 使用枚举定义全局状态，各任务根据状态执行不同行为：

```
STATE_SETTING ──(长按按钮)──▶ STATE_START_LINE_FOLLOW
     ▲                              │
     └────────(循迹完成后)──────────┘
```

- `blinkTask` 根据状态闪烁 LED：设置状态闪 N 次，运行状态常亮
- `mainTask` 根据状态决定是否启动循迹

## 6. 信号-槽 (Signal-Slot) 模式

[irq.c#L64-L68](Sources/irq.c#L64-L68) 和 [main.cpp#L145-L185](Sources/main.cpp#L145-L185) 实现了类 Qt 的信号槽机制：

```cpp
// 中断中只设置标志位（信号）
void ButtonPressed_Callback() { flagButtonPressed = 1u; }

// 任务中轮询处理（槽函数）
static void slotFunctionTask(void *arg) {
    if(flagButtonPressed) {
        // 区分短按/长按：每100ms检查一次，超过1s算长按
        for(i=0;i<10;i++) {
            if(flagButtonPressed || DL_GPIO_readPins(...)) break;
            vTaskDelay(pdMS_TO_TICKS(100));
        }
        if(i==10) slotFunction_ButtonHolded();
        else      slotFunction_ButtonClicked();
    }
}
```

**思想**：ISR 极短（只置位），业务逻辑在任务中执行，保证中断响应速度。

## 7. FreeRTOS 任务分解

| 任务 | 优先级 | 周期 | 职责 |
|------|--------|------|------|
| `motorControlTask` | 定时器中断触发 | 100Hz | 电机速度 PID + PWM 输出 |
| `mainTask` | -1 | 事件驱动 | 启动循迹 |
| `slotFunctionTask` | -2 | 10ms 轮询 | 按钮事件处理 |
| `blinkTask` | -2 | 状态驱动 | LED 状态指示 |
| `printLogTask` | -3 | 5s 周期 | 串口日志打印 |

**思想**：按**实时性要求**分层——最硬实时的（电机控制）用定时器中断，次实时的（UI）用高优先级任务，非实时的（日志）用低优先级。

## 9. C++ 在裸机/RTOS 嵌入式中的应用

项目大量使用 C++ 特性于 MCU：

- **类封装**：`PID` 类（位置式/增量式双模式）、`Motor` 类（聚合 PID）
- **构造函数参数推导**：`PID(PID_type, kp, ki, kd, 限幅...)` 用 `NAN` 表示不限幅
- **`[[nodiscard]]` 属性**：`PID::calc()` 返回值不应被忽略
- **`extern "C"`**：[main.cpp#L73-L76](Sources/main.cpp#L73-L76) 保证 C 语言中断处理函数能调用 C++ 代码
- **`__STATIC_INLINE`**：性能敏感的传感器读取用内联

## 10. 死推算 (Dead Reckoning) 定点停车

[main.cpp#L334-L342](Sources/main.cpp#L334-L342) 通过前两个弯角的时间戳推算最终停车所需的循迹周期：

```cpp
// t0=启动时刻, t1=第一个角, t2=第二个角
// 利用一圈四角的时间关系估算剩余路程
uint8_t finalCycles = (t2 + t0 - 2*t1 + 300) / 20;
```

结合最后的**斜坡减速停车**（45 步从占空比 45 逐步降到 0），避免急刹导致的车身晃动。
